genetic.js

This is a small library to explore using genetic algorithms. They are fun to use and can sometimes be used to solve interesting problems. https://en.wikipedia.org/wiki/Genetic_algorithm

Features

  • Use on the browser || in node.js programs
  • Built-in selection functions, crossover functions, mutation functions

Demos

Evolve text toward target http://liamgriffiths.github.io/geneticjs/examples/helloworld

Evolve a program that can find a composition of functions that takes an input number and gives an output number.

node examples/functions/functions.js
